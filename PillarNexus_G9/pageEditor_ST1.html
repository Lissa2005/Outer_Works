Technical Description

Splash Screen

The splash screen features a full-screen introduction with a semi-transparent overlay to ensure text readability. It displays a welcome message, group members, and a logo. A fade-out transition is triggered automatically or via the "Page Editor" button to reveal the main content smoothly.

Volunteer Page

The Volunteer page highlights opportunities supporting SDG 9 using card layouts. Each card includes an image, title, description, and a "Join Now" button, styled with Flexbox for a responsive grid. A dropdown filter allows users to sort opportunities, and user reviews plus a feedback form encourage interaction.

Content Page

The content page is organized into sections using <section> tags. Internal navigation is implemented via anchor tags and id attributes, and a "Go to Top" button is styled with position: fixed for easy access.

Challenges and Lessons Learned

It takes significant time to design a clean, professional website while maintaining functionality. I learned to manage my time better, especially when combining Flexbox layouts and responsive design, and to improve my overall front-end development workflow.

Accessibility

All images include alt text for screen readers. Semantic HTML elements like <nav> and <main> improve structure and accessibility. Buttons use descriptive labels, and color contrasts were checked for readability and accessibility compliance.

Compliance

All pages follow Janet web publishing guidelines, ensuring accessibility, responsive design, validated HTML/CSS, and no copyrighted material. Validation was performed using the W3C Validator to confirm compliance.
